Ayuda en la aplicación



Módulos Principales

groupPacientes
En este apartado podra encontrar toda la informacion referente a los pacientes que tiene asociados, podra monitorear a los pacientes de una manera síncrona (si el equipo lo permite) como asíncrona, podra asignarle equipos médicos, ver los registros asociados al equipo, históricos, e incluso añadir y eliminar pacientes del sistema.
perm_device_informationEquipos Médicos
En este apartado podra encontrar toda la informacion referente a los equipos médicos, podra editar la información propia del equipo y tambien información relacionada a los criterios de conexión con el equipo remoto, e incluso añadir y eliminar equipos del sistema.
settingsVariables Clínicas
En este apartado podra encontrar toda la informacion referente a las variables clínicas que defina, este apartado tiene la finalidad de establecer rangos de parámetros junto a las variables para poder evaluar los equipos de monitoreo con un nivel mas de flexibilidad ajustada por los médicos, podra editar la información de las variables, asociar equipos médicos previamente definidos, e incluso asociar pacientes con rangos particulares para poder evaluar detalles puntuales, e incluso asociar y eliminar equipos del sistema; asimismo, estas variables pueden crearse y eliminarse del sistema.
phonelink_setupParámetros
En el apartado de parametros generales podra visualizar información con la que el sistema se retroalimenta, podra establecer tipos de equipos medicos, modulos de visualización y modulos de deteccion de irregularidades en los registros médicos, todo esto asociado a los equipos medicos que esten en el sistema y los próximos que se vayan a registrar en el mismo; en este apartado se puede editar, crear y eliminar parámetros


Acciones

info_outlineHistóricos
Acción para visualizar los históricos de usuario
addNuevo
Acción para crear elementos nuevos
Acción para ejecutar el crear/editar un elemento en el sistema
Acción para volver a la vista de listado del módulo
remove
Acción para eliminar elementos existentes
edit
Acción para editar elementos existentes
settings_remote
Acción para poder consultar algun equipo de manera síncrona (si lo permite) ejecutando el módulo de visualización interno del equipo
visibility
Acción para poder visualizar los registros diferidos con un modulo de visualización asociado al equipo medico y un modulo de detección de irregularidades
done
elemento que indica que no existe ninguna irregularidad en el registro médico asociado
warning
elemento que indica que existe algun tipo de irregularidad en el registro médico asociado
